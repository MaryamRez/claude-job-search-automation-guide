# Claude Cowork Job Search Automation Guide

A comprehensive, step-by-step guide for setting up automated job searching using Claude Desktop's Cowork feature.

## ✨ Features

- **Complete Automation Setup**: 6-step process from connector installation to daily workflow
- **Smart Job Filtering**: AI-powered scoring and ranking system
- **Resume Integration**: Reduces setup questions by analyzing your background
- **Cover Letter Generation**: Automated drafts for top job matches
- **Modern UI**: Premium design with responsive layout
- **Mobile Optimized**: Works perfectly on all devices

## 🎯 What This Guide Covers

1. **Connector Installation** - Indeed and Apify setup
2. **Project Creation** - Cowork workspace configuration  
3. **Criteria Definition** - Personalized job requirements
4. **Scoring System** - Weighted evaluation framework
5. **Automation Setup** - Daily search and ranking
6. **Workflow Optimization** - Processing and application tips
